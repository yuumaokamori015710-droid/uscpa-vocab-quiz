# USCPA Vocab Quiz

USCPA学習者向けの英単語4択クイズアプリです。単語クイズに加えて、Abitus教材を前提にしたUSCPA全体の学習時間・進捗管理もできます。

## Features

- FAR / AUD / REG / BAR の英単語4択クイズ
- 学習履歴、正答率、Streak、苦手単語管理
- 友人ランキングのモック表示
- Dark / White テーマ切替
- Abitus教材別のUSCPA全体学習ペースメーカー
- 自学時間ログ、目標日、目標勉強時間、週次必要時間の管理

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Zustand

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000.
